dynplay
========

Spotify App that uses Echo Nest dynamic playlists

Installation
============

1. ensure you have a ~/Spotify directory
2. clone this repo into ~/Spotify/dynplay
3. edit index.html to put in your own EN ID
4. Open Spotify
5. Navigate to spotify:app:dynplay
